# Castle-Duel
<b>Description: </b>Castle-Duel is a Vue JS Framework educational card game.<br>

It's implemented from Vue.js 2 Web Development Projects Book.

## Dependencies
1- Vue JS<br/>
2- TWEEN JS<br/>
3- Google Fonts<br/>


## Installation
Just download and run index.html .
